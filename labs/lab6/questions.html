¿Cuáles son las diferencias entre los posibles valores de la propiedad position?

Hay 5 diferentes valores para la propiedad position:
Static:
Cuando se habla de posicionamiento de elementos en HTML, static es el valor por default, incluso aunque no escribamos explicitamente "position: static" HTML lo trae incorporado de forma implicita.
Con static lo que se busca es mantener el elemento deseado en una posicion relativa al flujo del documento sin que se le pueda hacer modificacion para tratar de moverlo hacia otro lugar.
Relative:
Con la posicion relative se mantiene el elemento en la posicion correspondiente al flujo del documento HTML, pero en esta ocasion podemos haceer uso de la propiedades top, left, bottom, right para ajustar la posicion del elemento conforme a nuestra conveniencia.
Fixed:
La posicion fixed permite que un elemento se mantenga en un lugar fijo a pesar de que se haga scroll.
Como un buen ejemplo de esto eatan las barras de navegacion como en Facebook, que te acompanan a pesar de que sigas bajando para ver contenido.
Absolute:
Absolute le da al elemento la facultad de mantenerse en una posicion, solo que en esta ocasion no es relativa a la pagina, si no al elemento dentro del cual este contenido, aunque si no hubiera elemento, si lo haria respecto a la pagina.
Sticky:
Sticky es la combinacion de relative con fixed, pues el elemento se vuelve fixed despues de que el elemento llega a una posision de la pagina donde pueda convertirse en fixed, por ejemplo, despues de haber hecho scroll a media pagina.

¿Cuáles son los valores estándar para la propiedad visibility?

Hay tres valores estandar:
visible:
visible permite que el elemento se pueda ver.
hidden:
hidden esconde el elemento, pero el espacio que ocupa se mantiene intacto.
collapse:
collapse sirve para "borrar" columnas o filas en una tabla, lo que hace es que las oculta de vista y quita el espacio que ocupaban para que se utilice por otros elementos.

¿Qué es el zIndex y para qué sirve?

Z-index funciona para colocar elementos unos encima de otros, con esta propiedad definimos cual elemento va encima de otros.
