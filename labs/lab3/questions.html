Preguntas

¿Cuál es la diferencia entre Internet y la World Wide Web?


Que internet es la conexión de dispositivos que siguen un mismo protocolo (TCP/IP) y WWW es el servicio que nos permite ver las páginas que comparten/crean usuarios con la conexión a internet[12]

¿Cuál es el propósito de los métodos HTTP: GET, HEAD, POST, PUT, PATCH, DELETE?


Controlar los datos de diferentes formas, con los metodos se pueden hacer cambios en los datos, recuperarlos, borrarlos.

¿Qué método HTTP se debe utilizar al enviar un formulario HTML, por ejemplo cuando ingresas tu usuario y contraseña en algún sitio? ¿Por qué?


Post, porque no permite al cliente ver los datos en la URL (como usando get).

¿Qué método HTTP se utiliza cuando a través de un navegador web se accede a una página a través de un URL?


Get, solo carga la URL.

Un servidor web devuelve una respuesta HTTP con código 200. ¿Qué significa esto? ¿Ocurrió algún error?


El código 200 es una respuesta satisfactoria, significa que la solicitud fue procesada con éxito, no hay error alguno.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 404? ¿Por qué?


No, ese error está en el servidor.

¿Es responsabilidad del desarrollador corregir un sitio web si un usuario reporta que intentó acceder al sitio y se encontró con un error 500? ¿Por qué?


Si, probablemente alguna parte del sitio está mal estructurada o contiene errores y a consecuencia el servidor no sabe cómo manejar el problema.

¿Qué significa que un atributo HTML5 esté depreciado o desaprobado (deprecated)? Menciona algunos elementos de HTML 4 que en HTML5 estén desaprobados.


Que un atributo este despreciado significa que el trabajo con CSS resulta mas efectivo que trabajar con los atributos, algunos de los despreciados en HTML5 son: Rev, charset, shape, coords, longdesc, target, nohref, profile, version, name, scheme, archive, classid, codebase

¿Cuáles son las diferencias principales entre HTML 4 y HTML5?


Que algunas herramientas de HTML4 dejaron de ser utilizadas en HTML5, además de que se procura que el código este “limpio” en HTML5

¿Qué componentes de estructura y estilo tiene una tabla?


La tabla tiene filas y columnas, bordes y colores.

¿Cuáles son los principales controles de una forma HTML5?


Input, lista de selección, área de texto, botones de enviar y deshacer

¿Qué tanto soporte HTML5 tiene el navegador que utilizas?


En Google chrome el soporte debe ser alto, pues es uno de los navegadores mas utilizados en todo el mundo.

Sobre el ciclo de vida y desarrollo de los sistemas de información:


¿Cuál es el ciclo de vida de los sistemas de información?


Planificación, desarrollo, integración, mantenimiento, obsolescencia[13]

¿Cuál es el ciclo de desarrollo de sistemas de información?


Planificación, análisis de requerimientos y requisitos, diseño del sistema, implementación y prueba de unidades, integración y prueba del sistema, operación y mantenimiento.[13]

Como ingeniero de software ¿cuál es tu recomendación sobre el uso de !important en un CSS?


Si hay algun elemento que debe permanecer "estatico" a lo largo del sitio es recomendable escribir !important en el CSS para evitar la sobreescritura de esta intruccion.

Si se pone una imagen de fondo en una página HTML, ¿por qué debe escogerse con cuidado?


Hay que verificar que la imagen no intervenga con los elementos del resto de la pagina, dado que la imagen puede contener elementos que no permitan que la parte visual de la pagina no sea la optima.

Como ingeniero de software, ¿cuál es tu recomendación al elegir las unidades de un propiedad de estilo entre %, px y pt?


si el sitio es pequeno y no importan los pequenos detalles no importa tanto si se usa alguna de estas tres unidades, pero lo recomendable seria usar porcentaje para que las paginas conserven sus propiedades dimensionales.

¿Por qué el uso de una versión minimizada del CSS mejora el rendimiento del sitio?


Porque la version "editable" del CSS contiene espacios en blanco y saltos de linea, por lo que el archivo se vuelve mas pesado, entonces sin esos elementos, el CSS minimizado requiere de menor tiempo de lectura por parte de la maquina.
