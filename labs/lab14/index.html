Bloque de preguntas

¿Qué es ODBC y para qué es útil?

Open DataBase Connectivity es un estandar que hace posible el acceder a cualquier dato desde cualquier aplicación, sea cual sea el gestor de Base de Datos[1]

¿Qué es SQL Injection?

Es la accion de introducir codigo SQL con el fin de hacer modificaciones en la base de datos de nuestro sistema

¿Qué técnicas puedes utilizar para evitar ataques de SQL Injection?

Esto es posible de prevenir haciendo uso de funciones en php donde se descompongan los caracteres de posibles queries de SQL; igualmente es buena practica mantener restricciones para los inputs de usuario y eliminar los procedimientos almacenados no utilizados[2]
