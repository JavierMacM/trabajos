¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

getDate(): Retorna el dia del mes
getHours(): Retorna la hora (de las 0 a las 23)
getMinutes(): Retorna los minutos
getTime(): Retorna la cantidad de milisegundos trasncurridos desde el 1/1/1970 hasta la fecha establecida
now(): Hace lo que getTime() pero desde la fecha actual
toJSON(): Retorna la fecha en una cadena en formato JSON

concat(): Sirve para unir n cantidad de arreglos
entries(): Retorna pares del arreglo (posicion, valor)

¿Cómo se declara una variable con alcance local dentro de una función?

Para que una variable sea local se tiene que declarar dentro de la funcion en la que se necesita.
Ejemplo:
var x;
function blabla(){}

Para que una variable sea global se tiene que declarar por fuera de la funcion.
Ejemplo:
function blabla(){ var x; }

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Que la variable puede ser alterada y entonces si otras funciones hacen uso de la variabloe puede haber resultados no deseados, ademas si en la funcion se creara una variable local pero con el mismo nombre puede haber fallos porque puede que se trabaje con la variable que no querramos.

¿Qué es material design?

Es un lenguaje visual que aplica las reglas generales del buen diseno para el desarrollo de aplicaciones web, sin comprometer la experiencia de usuario sea cual seas el navegador en el que se utilice la aplicacion.
