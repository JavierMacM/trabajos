Aparte de los arreglos $_POST y $_GET, ¿qué otros arreglos están predefinidos en php y cuál es su función?

  • PUT: se utiliza para enviar datos a un servidor para crear / actualizar un recurso.
  • HEAD: es casi idéntico a GET, pero sin el cuerpo de respuesta.
  • DELETE: borra el recurso especificado.
  • OPTIONS: describe las opciones de comunicación para el recurso objetivo.

¿Qué es XSS y cómo se puede prevenir.

Los ataques de secuencias de comandos entre sitios (XSS) son un tipo de inyección, en la que las secuencias de comandos malintencionadas se inyectan en sitios web benignos y de confianza. Los ataques XSS ocurren cuando un atacante usa una aplicación web para enviar código malicioso, generalmente en forma de un script del lado del navegador, a un usuario final diferente. [3]
